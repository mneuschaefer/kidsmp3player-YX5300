# kidsmp3player-YX5300

A simple Arduino-based kids MP3 player built around the YX5300 audio module.

> [!NOTE]
> This is a small hobby hardware prototype and is kept online for reference. It is not actively maintained.

## What It Does

This project controls a YX5300 MP3 module from an Arduino to build a child-friendly audio player.

Depending on the connected buttons and wiring, the player can be used to:

- play audio tracks from an SD card
- skip to the next or previous track
- pause and resume playback
- adjust volume
- create a simple, robust audio device without a display or complex interface

## What It Is Good For

This project is useful as:

- a starting point for a DIY kids audio player
- a reference for working with the YX5300 MP3 module
- an Arduino example for simple button-based audio control
- an experiment for building custom offline audio devices for children

A setup like this can be interesting if you want a very simple audio player for music, stories, or spoken content without needing a phone, tablet, or internet connection.

## Hardware

This prototype is based on:

- Arduino
- YX5300 MP3 audio module
- SD card with audio files
- buttons for playback control
- speaker or amplifier output, depending on your setup

## Repository Contents

- `kidsmp3player-test.ino`  
  Main Arduino sketch for testing and controlling the player.

## Status

This repository contains an early prototype / experiment. It may need adaptation for your exact hardware setup, wiring, and desired controls.
